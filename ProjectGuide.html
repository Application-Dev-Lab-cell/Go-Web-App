Go-Web-App

Complete DevOps Implementation in one project (CICD Using Docker and Kubernetes)

Git + Docker + Kubernetes + Helm Application + GitHub Actions + Argo

Step 1: Install Visual Studio

Install Visual Studio Application for coding.

Step 2: Install Git and Go

Download and install:

  • Git for Windows
  • Go Language
git clone https://github.com/Application-Dev-Lab-cell/Go-Web-App.git

Step 3: Install Docker Desktop

Create a Dockerfile and build:

docker build -t <docker-username>/go-web-app:v0 .

Run the container:

docker run -p 8080:8080 -it nkr0209/go-web-app:v0

Access: http://localhost:8080/courses

Step 4: Install Kubernetes Tools

  • Kubectl
  • eksctl
  • AWS CLI
aws configure

Use cluster.yaml to create your EKS cluster.

Step 5: Deploy Application

kubectl apply -f k8s/manifests/deployment.yaml
kubectl apply -f k8s/manifests/service.yaml
kubectl apply -f k8s/manifests/ingress.yaml

Check pods and services:

kubectl get pods
kubectl get svc
kubectl get ing

Step 6: Install Helm

helm version
helm install go-web-app ./go-web-app-chart

Helm will deploy your app as a chart with templates for Deployment, Service, and Ingress.
